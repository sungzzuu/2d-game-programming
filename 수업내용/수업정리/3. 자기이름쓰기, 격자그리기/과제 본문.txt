재귀 호출의 개념

함수 내부에서 함수가 자기 자신을 또다시 호출하는 행위. 재귀 호출은 자기가 자신을 계속해서 호출하고, 호출이 중단되도록 하는 조건의 명령문을 만날 때까지 반복되는 것이다.

 

재귀 호출의 활용 예

팩토리얼 값을 재귀 호출을 통해 구할 수 있다

 

def factorial(n):

if n== 1:

return 1

return n * factorial(n-1)

 

print(factorial(5))
